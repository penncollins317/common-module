# mxzero-common-project

`mxzero-common-project` 是一个用于沉淀和复用 **公共业务能力** 的项目，旨在为企业级应用提供通用的基础模块和服务支持。  
项目包含了常见的 **用户、认证、安全、支付、文件存储、消息通信、智能交互** 等业务场景，开发者可以直接使用或在此基础上扩展，减少重复造轮子。

---

## ✨ 项目目标

- 提供可复用的 **公共业务模块**
- 统一 **基础能力**（认证、存储、支付、消息等）
- 支持 **快速集成** 到新项目
- 作为 **微服务项目的底座**，方便扩展

---

## 🛠 技术特点

- **Spring Boot / Spring Cloud** 微服务架构
- **Gradle** 构建管理
- **PostgreSQL** 数据库支持
- 多种 **安全认证机制**（JWT、OAuth2、API Key、微信扫码等）
- 支持 **AI 能力**（对话、语音合成）
- 内置示例，便于学习和二次开发
